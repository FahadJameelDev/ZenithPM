# Zenith PM – Enterprise Project Management System

## Overview
Zenith PM is an Enterprise Project Management System developed using ASP.NET Core MVC and SQL Server. It helps organizations manage projects, tasks, departments, users, and attendance through a secure role-based system.

## Technologies Used
- ASP.NET Core MVC (.NET 10)
- React
- C#
- SQL Server
- Entity Framework Core
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- LINQ
- Git & GitHub

## Features
- Authentication & Authorization
- Role-Based Access Control (RBAC)
- Project Management
- Task Management
- Department Management
- Attendance Management
- Multiple Dashboards
- Responsive UI
- Clean Architecture

## Development Tools
- Visual Studio 2022
- SQL Server Management Studio (SSMS)
- Git & GitHub
